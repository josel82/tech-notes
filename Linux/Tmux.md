# Tmux

## Bash commands

`tmux` Start a new session `tmux attach` Attach to an existing session `tmux list-sessions` List existing sessions

## tmux commands

`crtl + b` Prefix key

Instructions:

| `d`           | Detach                              |
| ------------- | ----------------------------------- |
| `%`           | Vertical split                      |
| `"`           | Horizontal split                    |
| `left arrow`  | Move left through horizontal split  |
| `right arrow` | Move right through horizontal split |
| `up arrow`    | Move up through vertical split      |
| `down arrow`  | Move down through vertical split    |
| `x`           | Close session (paint)               |
| `c`           | Create a new window                 |
| `&`           | Close a window                      |
| `n`           | Go to next windows                  |
| `p`           | Go to previous window               |
| `[`           | Copy mode (Scroll)                  |
| `,`           | Name a window                       |
| `s`           | Session navigator                   |
| `,`           |  Rename session                     |
