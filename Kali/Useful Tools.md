## Mac Changer 
Allows you to change the Kali Linux Host's MAC address
- List options
```bash
macchanger --help
```

- Show MAC address of an interface
```bash
macchanger -s eth0 
```

- Change MAC address to ramdom value
```bash
macchanger -r eth0
```
